# Плагины

Плагины — механизм расширения osysHome.  
Этот документ описывает только общие принципы жизненного цикла и управления плагинами без инструкций по конкретным плагинам.

---

## Как загружаются плагины

При старте osysHome:
1. Сканирует каталог `plugins/`
2. Рассматривает каждую подпапку как кандидата в плагин
3. Проверяет статус активации в базе данных
4. Импортирует и инициализирует активные плагины

Неактивные плагины пропускаются.

---

## Управление плагинами в UI

Страница **Admin → Modules** позволяет:
- Смотреть список установленных плагинов
- Включать и отключать плагины
- Переходить в страницы настроек плагинов

> Для применения изменений жизненного цикла может потребоваться перезапуск.

---

## Установка плагина (общая схема)

1. Поместите папку плагина в `plugins/`
2. Убедитесь, что есть `__init__.py` и класс плагина
3. Установите зависимости (если есть `requirements.txt`)
4. Перезапустите osysHome
5. Включите и настройте плагин через **Admin → Modules**

---

## Где искать документацию по плагинам

Документация по конкретным плагинам поддерживается в самих плагинах:
- `plugins/<PluginName>/README.md`
- `plugins/<PluginName>/README.ru.md`
- при необходимости — `docs/` внутри папки плагина
